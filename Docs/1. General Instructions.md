## How to read the spreadsheet

1. The columns from A to F were added to better improve the project development (it works as a backlog).
2. Each row represents a task with its due prompt. 
3. If a task required more than one prompt, its result was properly split in the overview sentiment column.
4. In order to evaluate only agent tasks, filter column F (Agent Use) to display only value YES.
5. Some observation rows are important, especially those that have colors.

## Tips for the test

1. To avoid adding questions one by one and improve your test, you can import them using a CSV file. Check the How to import questions file. 
2. There's a CSV available in the Datasets folder.
